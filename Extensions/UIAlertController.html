  • init(whatsNew:versionStore:itemToText:onComplete:)

    Optional convenience initializer with WhatsNew and WhatsNewVersionStore. Checks if WhatsNewVersionStore has Version. If so initializer will return nil as this Version has been already been presented. If Version is unavailable initializer will return initialized UIAlertController and will save Version after completion
